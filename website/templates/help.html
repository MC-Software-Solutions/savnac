API token

  1. Log into canvas.
  2. Navigate to Account -> Settings using the navbar on the left of the screen.
  3. Scroll down and press the New Access Token button.
  4. Write anything in the purpose field and click Generate Token.

You can also refer to the official guide from Canvas

Organization domain

Your organization domain is just the domain name of the site you use for canvas. For example, browardschools.instructure.com is a valid domain name. Do not include the http:// or https:// scheme and do not include any slashes.
